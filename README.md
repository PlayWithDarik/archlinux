# ArchLinux

Скрипт для установки ArchLinux

pfi - git.io/JtvPD |
install - git.io/JtvPw |
aur - git.io/JD1Vs

# Установка
 ```bash
   pacman -Sy wget
   wget git.io/JtvPD && sh JtvPD
   ```
# AUR
```bash
   sudo pacman -Sy wget git
   wget git.io/JD1Vs && sh JD1Vs
   ```
# Это уничтожит безвозвратно все файлы на /dev/sdx
```bash
   wipefs --all /dev/sdx   
   ```
